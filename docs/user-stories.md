# User Stories

## Epic 1: Room Discovery

### US-001: View Available Rooms

As a visitor,  
I want to view the available hotel rooms,  
So that I can compare options before making a booking.

### US-002: View Room Details

As a visitor,  
I want to see room information such as name, description, price, and images,  
So that I can decide which room fits my needs.

---

## Epic 2: Booking Flow

### US-003: Select Booking Dates

As a visitor,  
I want to select check-in and check-out dates,  
So that I can book a room for a specific period.

### US-004: Submit Booking Information

As a visitor,  
I want to enter my personal details,  
So that I can complete a room booking.

### US-005: Receive Booking Confirmation

As a visitor,  
I want to receive a confirmation after submitting a valid booking,  
So that I know my reservation was created successfully.

### US-006: Validate Required Booking Fields

As a visitor,  
I want to see validation messages when required fields are missing or invalid,  
So that I can correct the information before submitting the booking.

---

## Epic 3: Contact Form

### US-007: Send Contact Message

As a visitor,  
I want to send a message through the contact form,  
So that I can contact the hotel with questions or requests.

### US-008: Validate Contact Form Fields

As a visitor,  
I want to receive clear validation messages when contact form fields are invalid,  
So that I can fix the form before submitting it.

---

## Epic 4: User Interface and Navigation

### US-009: Navigate Through the Website

As a visitor,  
I want to navigate through the website sections,  
So that I can find information quickly.

### US-010: Use the Website on Different Screen Sizes

As a visitor,  
I want the website to be usable on desktop and mobile screen sizes,  
So that I can access it from different devices.

---

## Epic 5: Accessibility

### US-011: Understand Visual Content with Assistive Technology

As a user relying on assistive technology,  
I want images and form elements to have meaningful accessible information,  
So that I can understand and use the website.

### US-012: Navigate Using Keyboard

As a keyboard-only user,  
I want to navigate interactive elements without a mouse,  
So that I can complete key actions independently.
